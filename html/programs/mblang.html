MBSE BBS: mblang (Language Compiler) Utility

Usage

mblang [language data file] [language source text]

 

Description

mblang compiles the source textfile to the language data file which is used by the mbsebbs program. You only need to use this program if you install a new language file. When you build the complete MBSE BBS package, this command is run automatically for you.
